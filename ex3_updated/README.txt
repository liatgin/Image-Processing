Ex3 Presumission script instructions:
1. extract the files to a folder with your exercise matlab functions
    files.
2. run ex3presub.m. Pay attention that in certain times you'll need
    to press a key in order to continue the script.
3. check that you got the massege: 'Pre-submission script done', if not
    you failed in one of the sections of the script. look at matlab console
    for figuring out which section failed.
